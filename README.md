# React-task

###### App uses

  - HTML5, CSS3     
  - JavaScript
  - React.js
  - Material-ui

# Start

  Clone project from github  

  ```sh
    $ git clone https://github.com/SargisK/React-task.git
  ``` 
  Install the dependencies

  ```sh
      cd react-task
      npm install
  ```

  Run the webpack dev server   

  ```sh
      npm start
  ```
